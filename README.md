## 🌟 نظرة عامة على المشروع (Project Overview)
نظام **Awsan NFC Global** هو بوابة دفع هجينة من الجيل الثالث (Web3-Ready)، تدمج تقنية الاتصال قريب المدى (NFC) مع شبكات البلوكشين (Blockchain) والعملات الرقمية والتقليدية في منصة واحدة فائقة الأمان.

### 🛠️ التقنيات المستخدمة (Tech Stack)

| التقنية | الوصف |
| :--- | :--- |
| **Python / Flask** | المحرك البرمجي لإدارة العمليات والـ Backend. |
| **Web3.py** | الربط مع شبكات العملات الرقمية (BTC, ETH, USDT). |
| **AES-256 Crypto** | بروتوكول تشفير عسكري لتأمين بيانات الـ NFC. |
| **SQLite / SQLAlchemy** | إدارة قواعد البيانات الضخمة للمحافظ والمتاجر. |
| **Bootstrap 5 (Dark Edition)** | واجهات مستخدم تفاعلية وعصرية. |

## 💎 المميزات الرائدة (Core Features)
*   ✅ **Hybrid Payment Engine:** دفع لحظي بالعملات المحلية (YER, SAR) والرقمية (Crypto).
*   ✅ **NFT Integration:** ربط الأصول الرقمية ببطاقات الـ NFC الفيزيائية.
*   ✅ **Global Connectivity:** جاهز للربط مع منصات Binance و MetaMask.
*   ✅ **Military-Grade Security:** نظام حماية ضد الاختراق والعمليات المزيفة.
*   ✅ **Automated Logic:** محرك ذكي لحساب العمولات وتحويل العملات آلياً.

## 🚀 طريقة التشغيل (Quick Start)
1. **تثبيت المكتبات:**
   ```bash
   pip install -r requirements.txt
   ```
2. **إعداد البيئة:** قم بإنشاء ملف `.env` ووضع مفتاح الأمان الخاص بك.
3. **التشغيل:**
   ```bash
   python awsan_nfc_2026.py
   ```
