## 🚀 نظرة عامة (Overview)
نظام **Awsan NFC** هو حل تقني متطور لإدارة المدفوعات عبر تقنية NFC، مصمم ليعمل في بيئات متعددة العملات (YER, SAR, USD, AED). يتميز النظام بلوحة تحكم احترافية وإدارة ديناميكية للموردين المعتمدين.

## ✨ المميزات التقنية (Core Features)
*   **Secure Authentication:** نظام دخول محمي بـ SHA-256 للمديرين فقط.
*   **Dynamic Vendor System:** إدارة فورية للمتاجر والموردين المعتمدين.
*   **Multi-Currency Engine:** معالجة العمليات بأربع عملات مختلفة بدقة عالية.
*   **Visual Dashboard:** واجهة مستخدم (UI) عصرية تدعم اللغة العربية بالكامل.
*   **Telegram Integration:** تنبيهات فورية للعمليات ومحاولات الاختراق.

## 🛠️ التقنيات المستخدمة (Tech Stack)
*   **Backend:** Python / Flask
*   **Database:** SQLite3 (Persistence Layer)
*   **Security:** Cryptography / Hashlib
*   **UI/UX:** Bootstrap 5 / Custom CSS3
