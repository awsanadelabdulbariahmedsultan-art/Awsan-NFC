## 🚀 نظرة عامة على المشروع (Overview)
نظام **Awsan NFC Vision 2026** هو محرك دفع متكامل يعتمد على تقنية اتصالات المدى القريب (NFC). صُمم ليكون الحل الأمثل للمتاجر والشركات التي تبحث عن السرعة، الأمان، والمرونة في التعامل مع العملات المتعددة (YER, SAR, USD, AED).

## ✨ المميزات التقنية (Core Features)
*   **🛡️ Secure Admin Shield:** لوحة تحكم محمية بنظام تشفير SHA-256 لمنع الدخول غير المصرح به.
*   **🏬 Dynamic Vendor Management:** نظام إدارة الموردين المعتمدين (Trusted Vendors) لضمان موثوقية نقاط البيع.
*   **💰 Multi-Currency Engine:** معالجة فورية للعمليات بأربع عملات دولية مع عرض الأرصدة بدقة.
*   **📊 Visual Analytics:** واجهة مستخدم (UI/UX) عصرية تدعم اللغة العربية (RTL) بالكامل.
*   **🔔 Real-time Notifications:** ربط مباشر مع Telegram لإرسال إشعارات العمليات وتنبيهات الأمان.
